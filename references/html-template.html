How to use
  • Order: build the checklist assets → generate high-risk prompts first.
  • Copy — the full standalone prompt, always English.
  • Click into any prompt — scene or asset — to edit. Local edits live only in this browser: to bake them into the file and refresh the translation — Export edits → paste into Claude.
  • RU — read-only translation of the prompt.
  • Statuses, keepers, notes — saved automatically.
  • Stuck? Open the Repair Guide. Scene changes — give this file back to Claude.
